El lenguaje de futuro

La programación (código) es el lenguaje que dominará la economía global. Nuestras coders aprenden diferentes lenguajes, HTML 5, CSS, PHP y Ruby que les permiten desarrollar un infinito número de productos y soluciones web.

Emprendimiento

El código y el emprendimiento van de la mano. Nuestras coders aprenden conceptos básicos del emprendimiento digital. El proyecto final de <Laboratoria> consiste en sacar adelante un Producto Mínimo Viable al mercado.

Aprender a aprender

Un principio básico de Laboratoria es que lo único que se necesita para ser una “crack” del desarrollo web es tener las ganas de explotar todos los recursos de autoaprendizaje que ofrece la web.

Crecimiento personal

En Laboratoria nos aseguramos que nuestras coders no sólo crezcan como desarrolladoras web, pero también como personas. Esto lo logramos a través de el apoyo de nuestras ONG aliadas.
